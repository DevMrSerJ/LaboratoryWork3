В папке "task" лежит текстовый файл для работы задания №1 ("task1"). Так как он меняется в процессе выполнения работы,
то лучше изменить его в данной папке ("...\Task") и потом положить его копию в папку "task\LabWork3". Также вы можете
изменять текстовый файл "task2" в папке "...\Task\LabWork3" для различных проверок работоспособности программы,
главное чтобы записи имели матричный вид и числа отделялись пробелом.
